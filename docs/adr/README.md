# ADR

ADR signifie `Architecture Decision Record`.

Ce dossier documente les decisions importantes prises dans le cadre du projet afin d'expliquer:

- pourquoi une solution a ete choisie;
- quelles alternatives existaient;
- quelles consequences cette decision entraine.
